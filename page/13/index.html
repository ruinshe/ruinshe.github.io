POJ2336 Ferry Loading II 动态规划

发表于 10月 10日, 2010 分类于 acm-icpc-solutions

好吧，今天开始正式学 DP，随便点了一个简单题，练了下手。。。

这道题转移方程不难，O(n2)O(n^2)O(n2) 的复杂度，可以 AC。

dp[i][j]dp[i][j]dp[i][j] 表示考虑跑了 iii 趟前 jjj 个车走完花的时间

先进行对时间排序(不知道为什么，第一次交没排序也 AC 了。。。。难道是原来就是有序的？)

i≤ni \leq ni≤n 时：dp[1][i]=a[i]+tdp[1][i] = a[i]+tdp[1][i]=a[i]+t
i>ni>ni>n时：dp[1][i]=∞dp[1][i] = \inftydp[1][i]=∞

至于趟数大于 111 的，有如下转移方程：

dp[k][i]=min⁡{max⁡(dp[i−1][j][j−1]+t,a[i])+t}dp[k][i] = \min\{\max(dp[i-1][j][j-1]+t, a[i])+t\} dp[k][i]=min{max(dp[i−1][j][j−1]+t,a[i])+t}

其中

max⁡(i−n+1,1)≤j<i\max(i-n+1,1) \leq j<i max(i−n+1,1)≤j<i

然后找 dp[i][m]dp[i][m]dp[i][m] 中最小的就可以了
